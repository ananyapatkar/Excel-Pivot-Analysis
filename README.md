Excel Analysis using Pivot Tables

## Objective
To analyze sales data using Excel Pivot Tables and generate meaningful business insights.

## Dataset
Superstore Sales Dataset

## Tools Used
- Microsoft Excel

## Work Done
- Cleaned the dataset and removed blank rows
- Converted data into Excel Table format
- Created Pivot Tables for:
  - Total Sales by Category
  - Sales by Region and Segment
- Applied conditional formatting to highlight top-performing regions
- Added slicers for interactive filtering
- Created a calculated column (Sales Contribution)
- Built Pivot Charts for visual analysis
- Summarized key insights
- Exported the final report as PDF

## Files Included
- Pivot_Report.xlsx – Excel pivot analysis
- Pivot_Report.pdf – Exported report


## Outcome
The analysis identified top-performing categories, regions, and customer segments, supporting data-driven decision making.
